# Landing Page - Maria Amélia Campos (Pré-Candidata PL-DF)

Este projeto é uma landing page profissional desenvolvida para a pré-campanha de Maria Amélia Campos à Câmara Federal em 2026.

## 🚀 Como Rodar

1.  **Baixe o arquivo ZIP** fornecido.
2.  **Extraia o conteúdo** em uma pasta no seu computador.
3.  Abra o arquivo `index.html` em qualquer navegador (Chrome, Edge, Firefox, etc.).

## 📂 Estrutura do Projeto

-   `index.html`: Estrutura principal do site.
-   `assets/css/style.css`: Estilização e design (cores do PL, responsividade).
-   `assets/js/main.js`: Interatividade (scroll suave, efeitos visuais).
-   `assets/img/`: Pasta contendo o logotipo e as fotos oficiais renomeadas.

## ✨ Destaques

-   **Design Responsivo:** Funciona perfeitamente em celulares, tablets e computadores.
-   **Identidade Visual:** Cores alinhadas ao Partido Liberal (Azul, Verde e Amarelo).
-   **Conteúdo Estratégico:** Focado em empreendedorismo, família e valores conservadores.
